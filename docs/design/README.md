# AURÓR — Referência Visual Somente

Este pacote deve ser usado apenas como referência visual para a Hub Financeira.

## Pode ser reutilizado

- paleta e tokens de cor;
- tipografia;
- escala tipográfica;
- espaçamentos e raios;
- sombras e movimento;
- superfícies, bordas e estados de interação;
- padrões visuais de botões, inputs, cards, badges, sinais e navegação.

## Não deve ser reutilizado

- contexto do Ecossistema Madan + Aurór;
- arquitetura de módulos do Ecossistema;
- agentes Aurora ou Dan;
- nomes de pessoas, empresas, fluxos ou objetos operacionais;
- textos de interface, exemplos de dados e taxonomias do produto anterior;
- componentes específicos como Copiloto, Solicitação, LayerChip, TaskItem e AgentRow;
- UI kits e telas prontas do Ecossistema.

## Diretriz de implementação

Trate estes arquivos como referência, não como dependência obrigatória. Adapte os componentes para a arquitetura e o vocabulário da Hub Financeira. Preserve a linguagem visual, mas crie componentes semânticos próprios do novo produto.

Os nomes históricos existentes em comentários ou variáveis de cor não definem significado de produto. Na Hub Financeira, as cores devem receber função semântica própria.
